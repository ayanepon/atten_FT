# Data directory

Place the MIMIR Wikipedia hard-split CSV files here:

```text
data/mimir_hardsplit/
  mimir_wikipedia_pt_member.csv
  mimir_wikipedia_ft_nonmember.csv
  mimir_wikipedia_unseen_nonmember.csv
  mimir_wikipedia_pt_ft_unseen_targets.csv
```

These files are generated by:

```bash
python src/train/train_mimir_wikipedia_hardsplit_lora.py --prepare-only
```

For exact cross-model comparison, reuse the same split CSVs for all models.
